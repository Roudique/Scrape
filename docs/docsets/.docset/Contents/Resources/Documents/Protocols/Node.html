Node

public protocol Node: Searchable

Instances of conforming types represent a document object model node that can be searched by an XPath or CSS selector and also provide the node’s underlying data in a number of ways.

In the HTML/XML DOM (Document Object Model), everything is a node: - The document itself is a document node; - All HTML/XML elements are element nodes; - All HTML/XML attributes are attribute nodes; - Text inside HTML elements are text nodes; - Comments are comment nodes.
